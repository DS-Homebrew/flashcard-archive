[instructions]
In order to play 3ds roms, you need to run 3DSLink_INSTALLER.NDS firstly in ds mode.
After running it successfully, please press HOME and return to the main menu.
If you fail, just try it once more.
Attention: you only need to run it once.

[教程]
为了进入3DS破解模式,请现在DS模式下运行3DSLink_INSTALLER.NDS.
运行成功之后,请按Home键直接退到主菜单.
注意:你只需要运行此文件一次即可.
