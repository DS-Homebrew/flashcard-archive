[English]
Note: This Upgrade package is XMENU 2.55 for 8G and 16G DSLinker both.
Important: Please make sure your firmware is 1.45 (enter USB MODE can check out). 

Q:How to upgrade the XMENU 2.55 of DSLinker?
1.Insert DSLinker Writer into slot 2 and insert DSLinker into slo 1.
2.Use USB cable connect to PC USB port directly. (Please do not use extension USB cable)
3.Open the power and enter USB MODE.
4.Copy "xmenu.dat" to DSLinker


Note: XMENU 2.55 can ONLY work with firmware 1.45.

Q: What's new for XMENU 2.55?
1. Fixed ROM 2107 display "thank" problem.
2. Fixed ROM 2207 can not save problem.
3. Added display storage space at UP screen.


[Chinese]

菜单程序v2.55
*  解决2107出现thank的问题
*  解决2207无法存档的问题
*  上屏幕增加剩余容量显示


1 升级方法
  *这个菜单需要U盘程序1.45版。
	(如果版本不对，会出现提示Need udisk version 1.45!)	
  *如果你当前的内核不是这个版本，就需要进行U盘程序的升级，请用以前的版本升级到1.45先。
  *如果你U盘程序已经是1.45版，就直接用新的文件覆盖NC上的xmenu.dat，升级完成。
	(如果你之前修改过global.ini关闭了dldi补丁，就要再次修改打开dldi)
